Lightbulb Favicon Generator

This page generates favicon files in different sizes from the lightbulb SVG icon.

16x16


32x32


48x48


180x180 (Apple)


Instructions:

  1. Right-click on each canvas and select "Save image as..." to download the favicon files
  2. Or use the download buttons to save the files
  3. Replace the existing favicon files in the public directory with these new ones
  4. The files should be named: favicon-16x16.png, favicon-32x32.png, favicon-48x48.png, apple-touch-icon.png
